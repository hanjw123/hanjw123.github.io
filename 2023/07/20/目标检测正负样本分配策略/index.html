目标检测正负样本分配策略

Label Assignment（标签分配）是目标检测算法在训练阶段的核心问题，涉及如何给特征图上的每个位置分配合适的学习目标，以及如何进行正负样本的分配。

一、学习目标的表示方法

Anchor Box类检测器

采用bounding box（x,y,w,h）表示目标，通过预测4维向量回归得到最终目标框。代表算法包括Faster RCNN。

Anchor Box构建要点：

  • 需要特定尺度（scale）和长宽比（aspect ratio）
  • 在特征图上以一定步长滑动生成
  • 将anchor box作为分类和框回归的基准框

Anchor Point类检测器

采用目标中心点，以及边框距中心点的距离或目标宽高或高斯热图表示目标。代表算法包括FCOS、CenterNet。

Key Point类检测器

采用图像中目标的边界点（如角点），再将边界点组合成目标框。代表算法包括CornerNet、RepPoints。

Set Prediction类检测器

代表作是DETR，将目标检测视为图像到集合（image-to-set）的问题。采用bounding box形式表示目标，将先验表示为object queries。

二、正负样本分配策略

Anchor Box类型检测器

通常采用IoU方法进行分配：

  • IoU低于0.4为negative（负样本）
  • IoU高于0.5为positive（正样本）
  • 其他为ignore（忽略样本）

代表算法：RetinaNet

Anchor Point类型检测器

不同算法表现形式不同：

  • CenterNet：通过高斯热图确定正负样本
  • FCOS：通过层级、区域和centerness分配正负样本

Key Point类型检测器

通过feature map bin和IOU进行正负样本区分。代表算法：RepPoints

Set Prediction类检测器

使用Hungarian Algorithm（匈牙利算法）实现预测值与真值的最大匹配，且是一一对应关系，无需NMS后处理。

三、Anchor Free与Anchor Based对比

类型 先验 代表算法 优势
Anchor Based Anchor Box Faster RCNN, RetinaNet 成熟稳定，检测精度高
Anchor Free Point FCOS, CenterNet 设计简洁，计算量小
