海思麒麟芯片处理器型号

本文汇总海思麒麟系列芯片处理器的主要型号和特点。

HI35xx系列（安防芯片）

HI3516

  • 产品定位：入门级安防SoC
  • CPU：ARM Cortex-A7/A9
  • GPU：Mali系列
  • 视频处理：支持1080p编码解码
  • 应用场景：网络摄像机、视频监控

参考链接：HI3516

HI3536

  • 产品定位：高性能安防SoC
  • CPU：ARM Cortex-A9
  • GPU：Mali-T628
  • 视频处理：支持4K编码解码
  • 应用场景：高端网络摄像机、NVR

参考链接：HI3536

HI3559

  • 产品定位：旗舰级安防SoC
  • CPU：ARM Cortex-A73 + Cortex-A53
  • GPU：Mali-G71 MP8
  • 视频处理：支持8K编码解码
  • AI能力：内置NNIE神经网络引擎
  • 应用场景：AI摄像机、智能安防

参考链接：HI3559

HI3519

  • 产品定位：中端安防SoC
  • CPU：ARM Cortex-A17
  • GPU：Mali-T760
  • 视频处理：支持4K编码解码
  • 应用场景：高清网络摄像机

参考链接：HI3519

HI3518

  • 产品定位：入门级安防SoC
  • CPU：ARM Cortex-A7
  • 视频处理：支持1080p编码
  • 应用场景：经济型网络摄像机

HiKey系列（开发板）

HiKey960

  • 产品定位：高端开发板
  • CPU：麒麟960 SoC
  • 架构：ARM Cortex-A73 + Cortex-A53
  • GPU：Mali-G71 MP8
  • 应用场景：Android开发、IoT原型验证

参考链接：HiKey960

麒麟系列（手机芯片）

麒麟960

  • 产品定位：高端手机SoC
  • CPU：4×Cortex-A73 + 4×Cortex-A53
  • GPU：Mali-G71 MP8
  • 工艺：16nm FinFET
  • 代表机型：华为Mate 9、P10

参考链接：麒麟960

麒麟970

  • 产品定位：旗舰手机SoC
  • CPU：4×Cortex-A73 + 4×Cortex-A53
  • GPU：Mali-G72 MP12
  • AI能力：内置NPU神经网络单元
  • 工艺：10nm FinFET
  • 代表机型：华为Mate 10、P20

参考链接：麒麟970

HI3815

  • 产品定位：物联网芯片
  • CPU：ARM Cortex-A7
  • 无线能力：支持Wi-Fi、蓝牙
  • 应用场景：智能家电、物联网终端

参考链接：HI3815

总结

海思麒麟芯片覆盖了多个领域：

  1. 安防领域：HI3516、HI3518、HI3519、HI3536、HI3559
  2. 手机领域：麒麟960、麒麟970等
  3. 物联网领域：HI3815等
  4. 开发板：HiKey960

不同型号针对不同市场定位，从入门级到旗舰级均有覆盖。
