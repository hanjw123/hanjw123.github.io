本文介绍在Java中加载动态链接库（.so文件）时需要注意的问题。

加载顺序问题

在Java中加载多个动态链接库时，加载顺序非常重要。

问题描述

如果先加载环境库（如OpenCV、CUDA等），当系统库中有其他版本的环境库时，可能会导致链接错误。
