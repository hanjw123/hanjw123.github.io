Python sort排序后返回索引方法

本文介绍Python中对数组进行排序后返回原始索引的方法。

方法一：使用numpy的argsort

argsort函数返回数组从小到大排序后的索引数组。

方法二：使用自定义类和sorted

定义一个包含数据和索引的类，使用lambda函数进行排序。

方法三：使用enumerate和sorted

更简洁的方式是使用enumerate函数。

方法对比

方法 适用场景 优点 缺点
numpy.argsort NumPy数组 简洁高效 需要NumPy依赖
自定义类 复杂对象 灵活定制 代码繁琐
enumerate+sorted 普通列表 无需额外依赖 性能略差

使用建议

  1. 处理NumPy数组时优先使用argsort
  2. 处理普通Python列表时使用enumerate+sorted
  3. 需要复杂排序逻辑时使用自定义类
