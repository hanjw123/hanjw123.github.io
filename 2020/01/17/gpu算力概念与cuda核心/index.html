GPU算力概念与CUDA核心详解

本文介绍GPU算力（Compute Capability）的概念和CUDA核心的相关知识。

参考链接：

  • https://developer.nvidia.com/zh-cn/cuda-gpus#compute
  • https://docs.nvidia.com/cuda/cuda-c-programming-guide/index.html#compute-capabilities

算力概念

GPU算力（Compute Capability）是NVIDIA对GPU硬件能力的一种标识，用版本号表示（如7.5、8.6等）。

算力版本组成

算力版本由主版本号和次版本号组成：

  • 主版本号：表示架构代际（如7.x对应Volta架构，8.x对应Ampere架构）
  • 次版本号：表示同一架构内的特性差异

主要架构对应的算力版本

架构 算力版本 代表GPU
Kepler 3.x GTX 7xx系列
Maxwell 5.x GTX 9xx系列
Pascal 6.x GTX 10xx系列
Volta 7.x V100, Titan V
Turing 7.5 RTX 20xx系列
Ampere 8.x RTX 30xx系列, A100
Hopper 9.x H100

CUDA核心

CUDA核心是GPU上执行计算的基本单元。

CUDA核心类型

  1. FP32核心：单精度浮点运算
  2. FP64核心：双精度浮点运算
  3. INT32核心：整数运算
  4. Tensor Core：张量运算（Volta及以上架构）

Tensor Core

Tensor Core是NVIDIA在Volta架构中引入的专用硬件单元，用于加速矩阵运算。

总结

  1. 算力版本决定了GPU支持的CUDA特性
  2. 不同架构的GPU有不同的算力版本
  3. Tensor Core是现代GPU的重要加速单元
  4. 编程时需考虑不同算力版本的兼容性
